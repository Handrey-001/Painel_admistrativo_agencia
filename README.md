Este é um sistema robusto de gestão desenvolvido para agências que necessitam de um controle centralizado de seus clientes e saúde financeira. A solução foca em **segurança**, **usabilidade** e **performance**.

## ✨ Funcionalidades Principais

* **📊 Dashboard Inteligente:** Visualização em tempo real do balanço financeiro e novos cadastros.
* **👥 Gestão de Clientes (CRUD):** Controle completo de clientes com interface intuitiva.
* **💰 Controle Financeiro:** Registro e monitoramento de entradas e saídas.
* **🔐 Segurança de Ponta:** Autenticação robusta e proteção contra ataques CSRF via Spring Security.
* **📱 Interface Responsiva:** UI moderna desenvolvida com Thymeleaf e CSS avançado.

---

## 🛠️ Tecnologias Utilizadas

O projeto utiliza o que há de mais moderno no ecossistema Java:

* **Backend:** Java 25 & Spring Boot 3
* **Dados:** Spring Data JPA & MySQL
* **Segurança:** Spring Security (Configurações customizadas)
* **Frontend:** Thymeleaf, HTML5 e CSS3
* **Versionamento:** Git & GitHub (Fluxo profissional)

## 🚀 Como Rodar o Projeto

Para testar o sistema localmente, siga os passos:

1.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/Handrey-001/Painel_administrativo_agencia.git](https://github.com/Handrey-001/Painel_administrativo_agencia.git)
    ```

2.  **Configuração do Banco de Dados:**
    * Crie um banco de dados MySQL chamado `painel_admin`.
    * Renomeie o arquivo `src/main/resources/application.properties.example` para `application.properties`.
    * Insira suas credenciais locais no arquivo renomeado.

3.  **Execução:**
    * Importe como um projeto Maven na sua IDE (VS Code, IntelliJ ou Eclipse).
    * Rode a classe `PainelAdminApplication.java`.
    * Acesse: `http://localhost:8080`
